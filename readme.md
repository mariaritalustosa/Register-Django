# Register Django

Projeto desenvolvido utilizando o framework Django para estudos e prática de desenvolvimento de aplicações web com Python.

## Tecnologias utilizadas

* Python
* Django
* HTML

## Estrutura do projeto

O projeto está organizado dentro da pasta principal:

```text
register_django/
```

A aplicação utiliza Python para a implementação da lógica do sistema em Django e HTML para a construção das páginas da aplicação.

## Como executar

### 1. Clone o repositório

```bash
git clone https://github.com/mariaritalustosa/Register-Django.git
```

### 2. Acesse a pasta do projeto

```bash
cd Register-Django/register_django
```

### 3. Execute o servidor Django

```bash
python3 manage.py runserver
```
